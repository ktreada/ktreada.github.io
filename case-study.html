Deliverables

3 Spot Color Label

The main focus of this project was to incorporate the use of only 3
colors in the design, Pantone 199 (red), Pantone 300 (blue) and
Pantone Black. The additionaly colors are created from overprints,
also referred to as built colors. These labels were printed on
pressure sensitive adhesive, and were diecut inline.

CMYK + Foil Package

Key features of this project include a 4 color process image,
adjusted to meet GraCol Standards, the application of cold foil,
dynamic barcodes, and the introduction of a dieline with both cut
and crease lines. A main challenge was including the relevant package
information while also making an aesthetically pleasing package.

Flexible Film Pouch

Because this design was printed on a transparent flexible film,
the main design concern was the creation of an opaque white layer
to ensure the CMYK inks would appear on the substrate. These
pouches were printed in reverse to that they would be right-reading
when rewound on the web.
